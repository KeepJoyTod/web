# Bug标题命名规范

## 1. 命名公式

```
[模块名] 操作/条件 + 问题现象
```

## 3. 标题模板

### 3.1 功能缺陷

```
[模块] 操作后的问题现象
[模块] 条件下的问题现象

示例：
[登录] 输入正确密码后提示密码错误
[购物车] 弱网下修改数量后合计金额未更新
[订单] 取消订单后库存未恢复
```

### 3.2 UI缺陷

```
[模块] 元素描述 + UI异常

示例：
[首页] Banner图片显示拉伸变形
[商品详情] 规格选择弹窗被底部导航遮挡
[购物车] 商品名称超过2行未截断
```

### 3.3 性能问题

```
[模块] 操作 + 性能指标异常

示例：
[首页] 首次加载耗时超过8秒
[搜索] 搜索结果页滑动卡顿
[订单] 订单列表加载白屏超过5秒
```

### 3.4 崩溃/闪退

```
[模块] 操作时 + 闪退/崩溃

示例：
[商品详情] 连续快速点击收藏按钮时闪退
[支付] 选择微信支付后APP崩溃
```

### 3.5 兼容性问题

```
[模块] [设备/系统] + 问题现象

示例：
[首页] iOS 16.5上TabBar图标缺失
[购物车] 小米13上合计金额被截断
```

### 3.6 数据异常

```
[模块] 数据描述 + 异常现象

示例：
[订单] 订单列表显示已取消订单的状态为"待付款"
[购物车] 商品数量为0时仍显示在购物车中
```

## 4. 标题质量评分

| 分数 | 标准 | 示例 |
|------|------|------|
| ⭐⭐⭐⭐⭐ | 模块+条件+现象，一看就懂 | [购物车] 弱网下修改数量后合计金额未更新 |
| ⭐⭐⭐⭐ | 模块+现象，信息充分 | [购物车] 修改数量后合计金额未更新 |
| ⭐⭐⭐ | 现象清晰，缺模块 | 修改商品数量后合计金额没有变化 |
| ⭐⭐ | 有方向但不具体 | 购物车金额有问题 |
| ⭐ | 模糊不可理解 | 购物车Bug |

## 5. 常见错误

| 错误类型 | ❌ 错误示例 | ✅ 正确示例 |
|----------|-----------|-----------|
| 太模糊 | [购物车] Bug | [购物车] 修改数量后合计金额未更新 |
